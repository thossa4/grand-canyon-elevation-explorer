Grand Canyon Elevation Explorer

This interactive web map introduces the Grand Canyon through a terrain-focused basemap. Users can zoom, pan, switch map styles, and click the reference point to begin exploring the landscape.

How to use this map

  • Drag the map to explore different parts of the canyon.
  • Use the + and - buttons to zoom in and out.
  • Open the layer control to switch between terrain and street views.
  • Click the marker to view the main reference point.

Data and basemap

This starter version uses online basemaps as the geographic layer for the project. It is designed as the minimum working version for Iteration 1 and can be expanded later with boundaries, viewpoints, trails, or additional elevation-related information.
